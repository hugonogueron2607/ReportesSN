Construir un FIS de tipo 2

Convierte el FIS de tipo 1 (fis1), en un FIS de tipo 2 (fis2)
fis2 = convertToType2(fis1);
El sistema Sugeno tipo 2 (fis2), utiliza funciones de membresía tipo 2 para las variables de entrada y funciones de membresía tipo 1 para las variables de salida.
Defina la huella de incertidumbre (FOU) para las funciones de mebresía de entrada. Para ello, establezca el factor de escala del función de membresía inferior para cada función de membresía. Para este ejemplo, establezca los valores de retardo de las funciones de membresía inferiores en 0.
scale = [0.2 0.9 0.2;0.3 0.9 0.3];
for i = 1:length(fis2.Inputs)
for j = 1:length(fis2.Inputs(i).MembershipFunctions)
fis2.Inputs(i).MembershipFunctions(j).LowerLag = 0;
fis2.Inputs(i).MembershipFunctions(j).LowerScale = scale(i,j);
end
end
Trace las funciones de membresía de entrada de tipo 2.
figure
subplot(1,2,1)
plotmf(fis2,'input',1)
title('Input 1')
subplot(1,2,2)
plotmf(fis2,'input',2)
title('Input 2')
El FOU añade incertidumbre adicional al FIS y produce una superficie de control no lineal.
figure
gensurf(fis2);
title('Control surface of type-2 FIS')
